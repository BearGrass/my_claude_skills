# 小说项目目录创建器

帮助用户创建标准化、结构化的小说项目目录，使创作过程井井有条。

## 工作流程

1. **获取项目信息**：询问用户小说项目名称（如未提供）
2. **创建目录结构**：按照标准模板创建完整的项目目录
3. **生成文档模板**：为每个阶段创建对应的记录文档
4. **输出说明**：向用户展示项目结构和各目录用途

## 目录结构

```
小说项目名/
├── README.md                    # 项目说明与进度追踪
│
├── 01-灵感创意/                  # 阶段一：创意孵化
│   ├── 创意记录.md              # 核心创意、灵感碎片
│   └── 灵感素材/                # 参考图片、文章、笔记
│
├── 02-背景设定/                  # 阶段二：世界观构建
│   ├── 世界观设定.md            # 时代、地理、社会规则
│   ├── 力量体系.md              # 如有修炼/魔法/科技体系
│   └── 设定素材/                # 地图、参考图等
│
├── 03-人物设定/                  # 阶段三：人物塑造
│   ├── 人物档案.md              # 所有人物的详细设定
│   ├── 主角小传.md              # 主角深度设定
│   └── 人物关系图.md            # 人物关系可视化
│
├── 04-故事骨架/                  # 阶段四：故事架构
│   ├── 故事大纲.md              # 整体故事框架
│   ├── 章节规划.md              # 分章细纲
│   └── 情节卡片.md              # 关键情节点
│
├── 05-小说初稿/                  # 阶段五：初稿写作
│   └── 章节/                    # 各章正文
│       ├── 第1章.md
│       ├── 第2章.md
│       └── ...
│
├── 06-结构修订/                  # 阶段六：结构优化
│   ├── 修订记录.md              # 修订问题与方案
│   ├── 修订稿/                  # 修订后的章节
│   └── 问题清单.md              # 待解决问题列表
│
└── 07-成品/                      # 阶段七：最终成品
    ├── 最终稿/                  # 润色后的完整稿件
    └── 投稿记录.md              # 投稿平台与反馈
```

## 执行步骤

1. 询问用户项目名称（如未在请求中提供）
2. 在用户指定位置或当前目录创建项目文件夹
3. 创建所有子目录
4. 生成 README.md 和各阶段的文档模板
5. 向用户展示创建结果和项目结构说明

## 使用示例

**用户输入**：创建小说项目《星河守望者》

**输出**：
- 创建文件夹 `星河守望者/`
- 生成完整目录结构
- 创建所有模板文件
- 显示项目创建成功信息
