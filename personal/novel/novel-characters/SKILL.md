# 小说人物设定生成器

根据创意和世界观设定，系统性地塑造鲜活的人物形象。

## 触发前提

此技能应在以下情况使用：
- 用户已有创意灵感（`01-灵感创意/`）
- 用户已有世界观设定（`02-背景设定/`）
- 用户想要创建或完善人物设定

## 工作流程

### 步骤1：读取现有素材

读取创意记录和世界观设定

### 步骤2：确定塑造重点（使用 AskUserQuestion 工具）

```
问题：您想优先塑造哪类人物？
选项：
- 主角：故事的核心人物
- 反派：与主角对抗的角色
- 配角：支持或阻碍主角的角色
- 全部人物：系统塑造所有核心人物
```

### 步骤3：核心人物塑造

先塑造主角，再扩展配角和反派

### 步骤4：用户确认（使用 AskUserQuestion 工具）

完成人物档案后，让用户确认：

```
问题：这个人物设定如何？
选项：
- 很满意，继续下一个人物
- 需要调整某些方面
- 重新生成这个人物
- 暂停，稍后继续
```

### 步骤5：生成人物档案

输出完整的人物设定文档

## 人物档案结构

### 主角档案（深度版）

```markdown
## 主角：[姓名]

### 基本信息
- **全名**：
- **年龄**：
- **性别**：
- **身份/职业**：
- **外貌特征**：身高、体型、面容特点、穿着风格、标志性特征
- **声音特点**：语调、说话方式

### 性格画像
- **表象性格**：别人眼中他是什么样的人
- **真实性格**：他真实的一面是什么
- **性格缺陷**：导致他陷入困境的内在弱点
- **性格优势**：帮助他克服困难的力量

### 内在驱动
- **核心欲望**：他最想要什么？（显性的目标）
- **深层需求**：他真正需要什么？（隐性的心理需求）
- **核心恐惧**：他最害怕什么？
- **核心信念**：他坚信什么？这个信念如何影响他的选择？

### 过去与现在
- **成长背景**：什么样的家庭和环境塑造了他
- **关键创伤**：过去对他影响最大的事件
- **当前困境**：故事开始时他面临的问题
- **日常状态**：普通一天的生活是怎样的

### 人物弧线
- **起点状态**：故事开始时的心理状态
- **转变契机**：什么事件促使他开始改变
- **挣扎过程**：他在改变过程中经历的困难
- **终点状态**：故事结束时的心理状态
- **成长代价**：为了成长他失去了什么

### 人物关系
- **最重要的人**：谁对他影响最大
- **最大对手**：谁是他最大的障碍
- **潜在盟友**：谁可能成为他的帮助者

### 独特元素
- **口头禅**：标志性的话语
- **习惯动作**：独特的行为习惯
- **爱好/兴趣**：展现性格的次要特征
- **秘密**：他不愿让人知道的事

### 关键场景
- **出场设计**：第一次出场如何展现性格
- **高光时刻**：最能展现角色魅力的场景
- **低谷时刻**：角色最脆弱的场景
- **转折时刻**：角色做出关键选择的场景
```

### 配角档案

```markdown
## 配角：[姓名]

### 基本信息
- **与主角关系**：
- **在故事中的作用**：推动情节/制造冲突/提供帮助/ comic relief
- **年龄/身份**：
- **外貌简述**：

### 性格特征
- **核心特点**：2-3个主要性格特征
- **说话风格**：独特的语言方式

### 人物功能
- **对主角的影响**：如何影响主角的决定或成长
- **情节功能**：在故事中承担什么作用

### 人物弧线
- **变化轨迹**：从什么状态到什么状态
- **关键转变点**：什么事件改变了TA

### 与主角的关系发展
- **初始关系**：
- **关系变化**：
- **最终关系**：
```

### 反派档案

```markdown
## 反派：[姓名]

### 基本信息
- **身份/地位**：
- **与主角的冲突**：利益冲突/理念冲突/情感冲突
- **危险性等级**：对主角构成的威胁程度

### 反派的逻辑
- **目标**：他想要什么？为什么想要？
- **动机**：为什么他认为自己的行为是合理的
- **信念**：他坚信什么价值观
- **底线**：他不会做什么？

### 反派的过去
- **成为反派的契机**：什么让他走上这条路
- **性格扭曲点**：哪里出了问题
- **可同情之处**：什么让读者能理解他

### 反派的能力
- **优势**：他的力量来源
- **弱点**：他的致命缺陷
- **资源**：他能动用什么

### 反派的策略
- **对抗方式**：如何与主角博弈
- **终极计划**：他的最终目标是什么
- **手段底线**：他愿意做什么/不愿做什么

### 人物弧线
- **发展轨迹**：
- **结局设计**：
```

## 人物塑造原则

### 主角塑造原则

1. **欲望与需求的矛盾**
   - 主角想要的（欲望）≠ 主角需要的（需求）
   - 这种矛盾是成长的动力

2. **缺陷即命运**
   - 性格缺陷导致困境
   - 克服缺陷才能成长

3. **过去解释现在**
   - 用过去的经历解释现在的行为
   - 创伤塑造了防御机制

4. **选择定义人物**
   - 在压力下做出的选择展现真正的性格
   - 每个选择都应该有代价

### 配角塑造原则

1. **功能明确**：每个配角都有存在的理由
2. **个性鲜明**：即使是小角色也要有记忆点
3. **关系推动**：通过配角展现主角的不同侧面
4. **适度发展**：配角有弧线但不喧宾夺主

### 反派塑造原则

1. **有自己的逻辑**：反派不认为自己是反派
2. **与主角镜像**：反派可能是主角的另一种可能性
3. **威胁真实**：反派的能力要匹配故事需求
4. **有可理解之处**：让读者理解（不必认同）他的动机

## 人物关系设计

### 关系类型矩阵

| 关系类型 | 戏剧功能 | 示例 |
|----------|----------|------|
| 镜像关系 | 展现另一种可能 | 主角vs反派 |
| 导师关系 | 传授知识/引导 | 导师→主角 |
| 挚友关系 | 支持/陪伴 | 主角+配角 |
| 恋爱关系 | 情感线/成长催化剂 | 主角+恋人 |
| 对抗关系 | 制造冲突 | 主角vs对手 |
| 家庭关系 | 根源/羁绊 | 主角与家人 |

### 关系发展弧线

```
初始状态 → 冲突/误解 → 共同经历 → 关系转变 → 新的平衡
```

## 人物数量建议

| 篇幅 | 核心人物数 | 主要配角数 | 次要人物数 |
|------|------------|------------|------------|
| 短篇(3万字以下) | 1-2 | 2-3 | 若干 |
| 中篇(3-15万字) | 2-3 | 3-5 | 若干 |
| 长篇(15万字以上) | 3-5 | 5-10 | 若干 |

## 使用示例

**用户输入**：帮我创建人物设定

**处理流程**：
1. 读取创意记录和世界观设定
2. 确定需要塑造的人物
3. 引导用户完善关键人物
4. 生成完整的人物档案

**输出**：完整的 `03-人物设定/` 目录内容
