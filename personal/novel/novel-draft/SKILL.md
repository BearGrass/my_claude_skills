# 小说初稿写作助手

根据故事大纲和人物设定，协助用户撰写小说初稿。

## 触发前提

此技能应在以下情况使用：
- 用户已完成故事大纲（`04-故事骨架/`）
- 用户想要开始撰写正文

## 工作流程

1. **准备阶段**：读取所有素材，确认当前进度
2. **写作引导**：帮助用户规划写作时段
3. **章节撰写**：协助撰写或优化章节内容
4. **进度追踪**：更新写作进度

## 写作模式

### 模式一：引导写作

适合不确定如何开始的作者。

```
1. 确认今天要写的章节
2. 回顾章节细纲和人物状态
3. 讨论开头、冲突、结尾
4. 作者写作，AI提供反馈
5. 完成后进行简短复盘
```

### 模式二：协写模式

适合需要AI参与创作的作者。

```
1. 确认章节目标
2. AI生成初稿段落
3. 作者修改和调整
4. 迭代优化
```

### 模式三：续写模式

适合已有部分内容的作者。

```
1. 读取已有内容
2. 分析风格和节奏
3. 提供续写建议或直接续写
4. 保持风格一致性
```

```markdown
- **场景情绪**：读者应该感到什么？
- **情绪变化**：情绪如何流动？

```


## 写作指导

### 开头技巧

```
### 强力开头类型

1. **动作开场**：直接进入行动
   "刀锋划破夜色的时候，李默还没意识到自己已经死了。"

2. **对话开场**：用有趣的对话吸引
   "如果你再不签字，我就要从这栋楼跳下去了。"
   "请便，窗户在那边。"

3. **悬念开场**：抛出疑问
   "没人知道她为什么会在那个雨夜出现在我家门口。"

4. **氛围开场**：建立独特氛围
   "这座城市已经三年没有见过太阳了。"

5. **人物开场**：展现人物特质
   "在所有人眼中，陈默都是一个不会说谎的人。他自己也是这么认为的。"

### 开头要避免
- 大段背景介绍
- 过于平淡的日常
- 没有冲突的对话
- 冗长的景物描写
```

### 对话技巧

```
### 对话原则

1. **每句对话都应有目的**
   - 推动情节
   - 展现性格
   - 制造冲突
   - 提供信息

2. **对话应有独特声音**
   - 不同人物说话方式不同
   - 用词、句式、语气要有辨识度

3. **用行动打断对话**
   - 不是两个脑袋在说话
   - 加入动作、表情、环境

4. **潜台词比表面话更重要**
   - 人物很少直接说出真意
   - 用沉默、转移话题表达

### 对话示例

❌ 平淡对话：
"你好吗？"
"我很好。"
"最近怎么样？"
"还行吧。"

✅ 有戏的对话：
"你看起来糟透了。"
"谢谢，你也很漂亮。"
"我是认真的。"
"我也是。"
```

### 场景技巧

```
### 场景构建要素

1. **感官细节**
   - 视觉：颜色、光线、动态
   - 听觉：声音、节奏、静默
   - 嗅觉：气味、空气
   - 触觉：温度、质感
   - 选择2-3种，不必面面俱到

2. **细节选择原则**
   - 只写对人物有意义的东西
   - 细节暗示情绪和主题
   - 用细节营造氛围

3. **节奏控制**
   - 紧张时刻：短句、快节奏
   - 情感时刻：细节、慢节奏
   - 变换节奏避免单调
```

### 人物塑造技巧

```
### 展现而非告知

❌ 告知：
他很紧张。

✅ 展现：
他的手指不停地敲击着桌面，目光时不时瞥向门口。

### 人物一致性

保持人物行为符合其性格：
- 建立人物行为档案
- 关键时刻的行为要有铺垫
- 行为变化要有理由

### 人物深度

通过矛盾展现深度：
- 外表强硬的人有脆弱时刻
- 善良的人有自私的瞬间
- 让人物做出意料之外但情理之中的选择
```

## 写作建议

### 克服写作障碍

```
### 卡文时

1. **跳过难点**
   先写后面的部分，回头再补

2. **降低标准**
   先写出来，不管好坏

3. **回顾大纲**
   确认这段的目标是什么

4. **改变方式**
   用口语先说一遍，再写下来

5. **休息一下**
   散步、听音乐，让潜意识工作
```

### 保持动力的方法

```
1. 设定每日最低字数（如500字）
2. 记录每日进步
3. 完成小目标后奖励自己
4. 与他人分享进度
5. 阅读同类型优秀作品获得灵感
```

## 使用示例

**用户输入**：帮我写第3章

**处理流程**：
1. 读取第3章的章节规划
2. 回顾前两章内容确保连贯
3. 引导用户确认本章要点
4. 协助写作或提供写作建议
5. 完成后更新进度

**输出**：完成的章节内容，保存到 `05-小说初稿/章节/第3章.md`
