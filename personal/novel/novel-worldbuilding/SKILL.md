# 小说背景设定生成器

根据创意灵感，系统性地构建小说的世界观设定。

## 触发前提

此技能应在以下情况使用：
- 用户已有创意灵感（记录在 `01-灵感创意/创意记录.md`）
- 用户想要创建或完善世界观设定

## 工作流程

1. **读取现有创意**：从项目目录读取已有的创意记录（如有）
2. **理解类型需求**：根据小说类型确定世界观侧重点
3. **引导式提问**：通过问题引导用户完善设定
4. **生成设定文档**：输出完整的世界观设定

## 世界观要素框架

### 基础框架（所有类型通用）

```
## 时代背景
- 时间设定：历史时期/现代/未来年代/架空纪年
- 时代特征：这个时代的社会风貌、技术水平、文化氛围

## 地理环境
- 世界范围：故事发生的世界有多大
- 核心区域：主要故事发生的地点
- 重要地标：具有特殊意义的地点
- 地理对故事的影响：环境如何影响人物和情节

## 社会结构
- 权力体系：谁在统治？如何统治？
- 阶级划分：社会阶层如何划分？
- 组织势力：有哪些重要组织/势力？
- 社会矛盾：潜在的冲突来源

## 核心规则
- 世界运行法则：这个世界最基本的规则
- 独特设定：区别于现实世界的特殊之处
- 代价与限制：设定的边界在哪里

## 文化风貌
- 价值观念：这个社会推崇什么？
- 风俗习惯：独特的习俗传统
- 语言特色：称谓、俚语、禁忌语

## 经济体系
- 资源分布：重要资源有哪些？
- 贸易流通：经济如何运作？
- 贫富差距：财富分配状况

## 历史脉络
- 重要历史事件：塑造当今世界的关键事件
- 历史遗留问题：影响当下冲突的历史因素
- 传说与谜团：待揭开的秘密
```

### 类型特化框架

#### 玄幻/奇幻类

```
## 力量体系
- 力量来源：灵气/魔法/神力/血脉...
- 修炼等级：从低到高的境界划分
- 突破条件：如何变强？代价是什么？
- 力量限制：为什么不是所有人都强大？

## 种族设定
- 主要种族：人族/妖族/魔族/神族...
- 种族特征：各种族的能力与弱点
- 种族关系：联盟/敌对/奴役...

## 神话传说
- 创世神话：世界如何诞生
- 神明体系：有哪些神？信徒是谁？
- 禁忌领域：不可触碰的禁区

## 珍稀资源
- 天材地宝：修炼资源
- 法宝设定：武器/法宝的等级与来源
- 资源争夺：围绕资源的冲突
```

#### 科幻类

```
## 科技水平
- 核心技术：改变世界的关键技术
- 技术限制：什么技术不可行？为什么？
- 技术影响：科技如何改变社会？

## 星际设定（如适用）
- 星际范围：人类探索的疆域
- 星球特点：重要星球的特色
- 航行方式：星际旅行的原理与代价

## 社会变革
- 新社会形态：技术带来的社会变革
- 人机关系：AI/机器人的地位
- 伦理困境：技术带来的新问题

## 未来历史
- 关键节点：从现在到未来的关键事件
- 文明危机：曾经面临的危机
- 未解之谜：科学无法解释的现象
```

#### 都市类

```
## 城市设定
- 城市特点：故事发生城市的特色
- 区域划分：富人区/贫民区/商业区...
- 城市问题：交通/住房/治安...

## 行业背景
- 核心行业：故事涉及的主要行业
- 行业规则：潜规则、行规
- 行业现状：机遇与挑战

## 社会议题
- 热点问题：与故事相关的社会问题
- 阶层流动：上升通道与障碍
- 时代焦虑：当代人的普遍困境

## 真实感营造
- 流行文化：音乐/电影/网络文化
- 生活细节：消费习惯/生活方式
- 时代印记：区分时代的关键元素
```

#### 悬疑/推理类

```
## 案件背景
- 案件类型：谋杀/失踪/诈骗...
- 侦破环境：警力/技术/资源
- 悬疑氛围：如何营造紧张感

## 隐藏世界
- 地下势力：黑帮/地下交易
- 秘密组织：幕后操控者
- 不为人知的角落

## 信息不对称
- 真相层：真相是什么
- 表象层：看起来是什么
- 揭示节奏：如何逐步揭开

## 心理设定
- 犯罪心理：反派的行为逻辑
- 创伤影响：事件对人物的影响
- 道德困境：善恶的灰色地带
```

#### 言情类

```
## 情感环境
- 恋爱观念：社会对爱情的态度
- 婚恋压力：来自家庭/社会的压力
- 时代特色：当代恋爱的特点

## 阻碍来源
- 身份差距：阶层/身份的差异
- 外部阻力：家庭/社会/竞争对手
- 内心障碍：过去的创伤/恐惧

## 浪漫元素
- 约会场景：故事发生的浪漫地点
- 情感符号：承载意义的物品/场景
- 甜蜜日常：增进感情的互动

## 成长环境
- 家庭背景：双方家庭的差异
- 成长经历：塑造性格的过往
- 未来选择：爱情与理想的冲突
```

## 引导问题

在创建设定时，向用户提问以下问题（根据类型选择）：

### 核心问题
1. 故事发生的时代和地点是什么？
2. 这个世界最独特的设定是什么？
3. 这个世界最大的矛盾/冲突是什么？
4. 主角在这个世界中处于什么位置？

### 深入问题
1. 普通人在这世界如何生活？
2. 权力掌握在谁手中？他们如何维持统治？
3. 这个世界有哪些禁忌？为什么？
4. 如果有超自然元素，它的代价是什么？
5. 主角想要改变这个世界的什么？

## 使用示例

**用户输入**：根据我之前的创意，帮我创建背景设定

**处理流程**：
1. 读取 `01-灵感创意/创意记录.md`
2. 根据创意类型选择框架
3. 引导用户补充关键信息
4. 生成完整的世界观设定

**输出**：完整的 `02-背景设定/世界观设定.md` 文件
