# Novel 创作工作流技能集

一套完整的小说创作辅助技能，覆盖从灵感到成品的完整流程。

## 快速开始

使用 `novel-workflow` 技能，一键引导完成整个创作流程：

```
/novel-workflow
```

或说："引导我写小说"

## 创作规范

所有创作活动遵循 `novel-standards` 规范：
- **文件存放**：`~/novels/` 目录
- **写作风格**：网文风格，适合短剧/动画/有声书改编
- **文字校对**：自动检查错别字和标点
- **内容边界**：适度情感描写，避免低俗
- **文笔标准**：简洁有力，画面感强

## 核心架构

### 三阶段工作流

```
┌─────────────────────────────────────────────────────────────────┐
│                     小说创作完整工作流                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  阶段一：筹备期                                                  │
│  ├── 项目创建 → 灵感创意 → 背景设定                              │
│  └── 人物设定 → 故事骨架                                        │
│                                                                 │
│  阶段二：创作期（novel-writing 控制）⭐                          │
│  ├── 分章节初稿写作                                             │
│  ├── 分层修订优化                                               │
│  └── 分层润色提升                                               │
│                                                                 │
│  阶段三：成品期                                                  │
│  └── 输出最终稿件                                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 写作控制台分层架构

`novel-writing` 提供分层处理，避免单次大规模操作耗时过长：

```
初稿分层：                    修订分层：
├── 场景层（500-2000字）      ├── 章节修订（快，5-10分钟）
├── 章节层（3000-8000字）     ├── 单元修订（中，15-30分钟）
├── 单元层（3-10章）          ├── 故事线修订（慢，30-60分钟）
└── 全文层                    └── 整体修订（最慢，1-2小时）

润色分层：
├── 语句润色（随修订同步）
├── 章节润色（修订后统一）
└── 风格统一（最终检查）
```

### 推荐写作节奏

```
快速迭代模式（推荐）：
写一章 → 快速修订 → 标记问题 → 下一章
完成一个单元后 → 单元级修订 → 继续
完成初稿后 → 故事线修订 → 整体修订 → 润色
```

## 使用指南

### 1. 安装技能

将技能目录复制到 Claude Code 的 skills 目录：

```bash
cp -r personal/novel/* ~/.claude/skills/
```

### 2. 开始创作

在 Claude Code 中按顺序使用：

```
1. /novel-project     # 创建项目目录
2. /novel-ideas       # 生成创意方案
3. /novel-worldbuilding  # 构建世界观
4. /novel-characters  # 创建人物设定
5. /novel-outline     # 创作故事大纲
6. /novel-draft       # 撰写初稿
7. /novel-revision    # 修订优化
8. /novel-polish      # 润色成品
```

### 3. 迭代创作

工作流支持灵活迭代：
- 可以随时回到前一阶段修改
- 每个阶段可以多次执行
- 后续技能会自动读取前置素材

## 技能详细说明

### novel-project - 项目创建
创建标准化的小说项目目录，包含各阶段文档模板，帮助有序管理创作全过程。

### novel-ideas - 创意生成
生成3-5个结构完整的小说创意方案，每个包含：标题、简介、主角设定、核心冲突、结局走向、亮点卖点。支持多种类型和关键词定向生成。

### novel-worldbuilding - 世界观构建
系统性地构建小说世界观，包括：时代背景、地理环境、社会结构、核心规则等。针对不同类型（玄幻/科幻/都市/悬疑/言情）提供特化框架。

### novel-characters - 人物塑造
塑造完整的人物形象，包括主角深度档案、配角设定、反派设计。涵盖人物内在驱动、成长弧线、人物关系等。

### novel-outline - 故事架构
构建完整的故事结构框架，支持三幕式结构、英雄之旅、悬疑结构、言情结构等多种模型。输出故事大纲和章节细纲。

### novel-draft - 初稿写作
协助撰写小说初稿，提供写作指导、技巧建议、进度追踪。支持引导写作、协写、续写等多种模式。

### novel-revision - 结构修订
对初稿进行深度结构分析，发现并修复情节漏洞、节奏问题、人物弧线问题等。系统性地记录修订过程。

### novel-polish - 润色成品
对修订稿进行文字层面的精细打磨，优化语言表达、提升阅读体验，最终输出成品稿件。
