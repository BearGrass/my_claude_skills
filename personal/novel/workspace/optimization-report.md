# Novel 技能描述优化报告

## 优化概述

**优化日期**: 2026-03-02
**技能数量**: 12 个
**优化目标**: 消除触发词冲突，明确技能边界

---

## 一、发现的问题

### 1. 触发词冲突

| 冲突触发词 | 涉及技能 | 问题说明 |
|------------|----------|----------|
| "开始写小说" | novel-workflow, novel-project | 用户可能只想要目录，但触发了流程引导 |
| "写初稿"/"开始写作" | novel-draft, novel-writing | 用户可能需要进度控制，但触发了单章写作 |
| "修订"/"润色" | novel-revision, novel-polish, novel-writing | 边界不清，不确定是否需要进度管理 |

### 2. 内容冗余

- `novel-project` 的目录结构定义与 `novel-standards` 重复
- `novel-draft` 的写作指导与 `novel-writing` 重叠

### 3. 技能层级不清

```
优化前：
├── novel-workflow（触发范围过宽）
├── novel-ideas
├── novel-project
├── novel-worldbuilding
├── novel-characters
├── novel-outline
├── novel-draft（与 writing 关系不清）
├── novel-writing（职责模糊）
├── novel-revision（与 writing 关系不清）
├── novel-polish（与 writing 关系不清）
├── novel-standards
└── novel-cover
```

## 二、优化方案

### 1. 明确技能层级

```
优化后：
novel-workflow ───────────────── 总入口（完整流程引导）
│
├── 筹备期专项
│   ├── novel-ideas ──────────── 灵感创意
│   ├── novel-project ────────── 创建目录
│   ├── novel-worldbuilding ──── 世界观设定
│   ├── novel-characters ─────── 人物设定
│   └── novel-outline ────────── 故事大纲
│
├── 创作期
│   └── novel-writing ────────── 写作控制台
│       ├── novel-draft ──────── 单章节写作
│       ├── novel-revision ───── 结构修订
│       └── novel-polish ─────── 文字润色
│
└── 成品期
    ├── novel-standards ──────── 规范查询
    └── novel-cover ──────────── 封面生成
```

### 2. 描述优化详情

#### novel-workflow

```yaml
优化前:
  触发词: "写小说"、"创作小说"、"开始写小说"、"引导我写小说"
  问题: 触发范围过宽，与其他技能冲突

优化后:
  触发词: "引导我写小说"、"怎么开始写小说"、"完整创作流程"、"从头写小说"
  改进: 明确为"全流程引导入口"，区分具体步骤
  边界说明: 如果用户只需要某个具体步骤，触发对应专项技能
```

#### novel-project

```yaml
优化前:
  触发词: "创建小说项目"、"新建小说项目"、"开始写小说"、"搭建小说目录"
  问题: "开始写小说"与 workflow 冲突

优化后:
  触发词: "创建小说项目"、"新建小说目录"、"初始化小说项目"、"搭建小说工作目录"
  改进: 删除"开始写小说"，专注于目录创建
  边界说明: 如果用户需要完整创作流程，触发 novel-workflow
```

#### novel-draft

```yaml
优化前:
  触发词: "写初稿"、"开始写作"、"写第X章"、"撰写章节"
  问题: "开始写作"与 writing 冲突

优化后:
  触发词: "写初稿"、"写第X章"、"撰写章节"、"开始写正文"、"帮我写这一章"
  改进: 删除"开始写作"，专注于单章节写作
  边界说明: 如果用户需要进度控制，触发 novel-writing
```

#### novel-writing

```yaml
优化前:
  触发词: "控制写作节奏"、"写作进度"、"分章节写作"、"分层修订"、"继续写作"
  问题: 与 draft/revision/polish 关系不清

优化后:
  触发词: "控制写作节奏"、"查看写作进度"、"继续写作"、"分层修订"、"分层润色"、"写作状态"
  改进: 明确为"写作控制台"，包含进度管理
  边界说明: 如果用户只是写某一章，触发 novel-draft
```
