# 小说创作工作流引导器

引导用户完成从灵感到成品的完整小说创作流程，全程遵循创作规范。

## 规范基础

**所有创作活动均遵循 `novel-standards` 规范**，包括：
- 文件目录：作品存放在 `~/novels/` 目录
- 写作风格：网文风格，适合短剧/动画/有声书改编
- 文字校对：自动检查错别字和标点符号
- 内容边界：适度情感描写，避免低俗
- 文笔标准：简洁有力，画面感强

详见 `novel-standards` 技能。

## 工作模式

### 模式选择（使用 AskUserQuestion 工具）

当需要让用户选择模式时，使用 AskUserQuestion 工具提供选项：

```
问题：请选择创作模式？
选项：
- 完整引导（推荐新手）：从零开始，完整走完所有步骤
- 快速启动：获取基本信息后直接创建项目
- 继续创作：检测已有项目并继续
- 直接写作：确认有大纲后直接进入写作
- 仅生成封面：读取已有设定生成封面
```

### 模式一：完整引导（推荐新手）
1. 规范确认 → 项目创建 → 灵感创意 → 背景设定 → 人物设定 → 故事骨架 → 写作控制台 → 质量验收 → 封面生成 → 输出成品

### 模式二：快速启动
1. 获取基本信息 → 在 ~/novels/ 创建项目 → 从指定步骤开始

### 模式三：继续创作
1. 检测 ~/novels/ 下的项目 → 读取进度 → 继续创作

### 模式四：直接写作
1. 确认有大纲 → 直接进入 novel-writing

### 模式五：仅生成封面 🆕
1. 读取已有设定 → 生成番茄小说封面

## 执行流程

### 步骤0：规范确认

```
┌─────────────────────────────────────────────────────────┐
│                    创作规范确认                          │
├─────────────────────────────────────────────────────────┤
│  📁 文件存放：~/novels/                                  │
│  ✍️ 写作风格：网文风格，适合改编                         │
│  📝 文字规范：自动检查错别字和标点                       │
│  🎭 内容边界：适度情感描写                               │
│  💡 文笔标准：简洁有力，画面感强                         │
└─────────────────────────────────────────────────────────┘

确认以上规范？[Y/n]
```

### 步骤1：项目创建
- 在 `~/novels/[小说名]/` 创建项目
- 生成标准化目录结构

### 步骤2-5：筹备期
- 灵感创意 → 背景设定 → 人物设定 → 故事骨架
- 每步都遵循 novel-standards 规范

### 步骤6：写作控制台
- 调用 novel-writing
- 实时规范检查（错别字、标点、尺度）

### 步骤7：质量验收
- 调用 novel-standards 质量检查清单
- 生成验收报告

### 步骤8：封面生成 🆕

**目标**：生成番茄小说风格封面

**执行**：
调用 `novel-cover` 技能生成封面

使用 AskUserQuestion 工具让用户选择：

```
问题：请选择封面操作？
选项：
- 使用此方案生成封面：按当前设计生成封面
- 调整设计方案：修改风格、色调等
- 查看其他风格选项：浏览不同封面风格
- 跳过封面生成：暂不生成封面
```

**封面规格**：
- 尺寸：600×800 像素（番茄小说标准）
- 必须包含小说标题文字
- 风格匹配小说类型

**输出位置**：
```
~/novels/[小说名]/成品/封面/
├── 封面-原始图.png
├── 封面-带标题.png
├── 封面-缩略图.jpg
└── 封面设计方案.md
```

### 步骤9：输出成品
- 保存到 `~/novels/[小说名]/成品/`
- 包含完稿和封面

## 注意事项

1. **规范是基础**：所有创作活动必须遵循 novel-standards 规范
2. **统一存放位置**：作品统一存放在 ~/novels/ 目录
3. **网文风格优先**：写作风格面向网文读者和多媒体改编
4. **实时规范检查**：写作过程中自动检查错别字、标点、尺度
5. **质量验收**：成品输出前必须通过规范检查
6. **封面必须**：封面必须包含小说标题文字，符合番茄小说规格
