# 小说创作规范

定义小说创作的各项标准，确保作品质量和一致性。

## 一、文件目录规范

### 存放位置

```
默认创作目录：~/novels/

目录结构：
~/novels/
├── [小说名]/
│   ├── 00-设定/
│   │   ├── 世界观.md
│   │   ├── 人物设定.md
│   │   └── 故事大纲.md
│   ├── 正文/
│   │   ├── 第1章.md
│   │   ├── 第2章.md
│   │   └── ...
│   ├── 存档/
│   │   └── [日期]-v[X]/
│   └── 成品/
│       └── [小说名]-完稿.md
```

### 文件命名规则

```
章节文件：第N章.md（如：第1章.md、第12章.md）
设定文件：[类型]设定.md（如：世界观.md、人物设定.md）
存档文件：YYYY-MM-DD-vN（如：2026-03-02-v1/）

禁止：
- 使用特殊字符
- 文件名过长
- 中英文混用命名
```

### 版本管理

```
每完成一个单元（5-10章）自动存档：
存档命名：存档/YYYY-MM-DD-v[N]/

重要节点存档：
- 初稿完成时
- 重大修订后
- 成品输出前
```

## 二、写作风格规范

### 网文风格特征

```
核心原则：易读、有画面感、节奏明快

1. 句式特点
   - 多用短句，避免长难句
   - 段落简短，一般不超过5行
   - 频繁换行，增加阅读节奏感

2. 叙述方式
   - 以动作和对话推动情节
   - 减少大段心理描写
   - 场景转换清晰明确

3. 语言风格
   - 口语化表达，贴近生活
   - 避免文言文和生僻字
   - 适当使用网络流行语（符合时代背景时）
```

### 多媒体改编适配

```
适合短剧改编：
├── 场景明确，易于拍摄
├── 对话为主，减少旁白
├── 人物动作具体可执行
└── 每章有独立的小高潮/悬念

适合AI短动画：
├── 画面感强，易于AI生成
├── 人物形象鲜明，特征明确
├── 场景描述简洁但完整
└── 动作连贯，逻辑清晰

适合有声书：
├── 对话自然，适合朗读
├── 避免过多视觉描写
├── 节奏有起伏，留有呼吸空间
└── 关键信息通过对话传达
```

### 章节结构标准

```
章节开头（100-300字）：
├── 快速进入场景
├── 或承接上章悬念
└── 不用大段背景介绍

章节主体（2000-5000字）：
├── 1-3个场景
├── 至少一个冲突点
└── 对话占比40%-60%

章节结尾（50-200字）：
├── 小悬念/钩子
├── 或情感落点
└── 引导阅读下一章
```

## 三、文字校对规范

### 错别字检查清单

```
常见错别字对照表：

易混淆字：
- 的/地/得：修饰名词用"的"，修饰动词用"地"，动词后用"得"
- 做/作：具体动作用"做"，抽象行为用"作"
- 在/再：地点/状态用"在"，重复/继续用"再"
- 已/以：完成用"已"，用/因为用"以"
- 象/像：比拟用"像"，形象用"象"（如：抽象、气象）

网文常见错误：
- 那儿/哪儿 → 哪儿（疑问）、那儿（指示）
- 俩/两个：口语可用"俩"，叙述用"两个"
- 什么的/甚么：统一用"什么"
- 甚至/甚而：统一用"甚至"

人名/地名一致性：
- 全文统一，避免同一人物不同写法
- 建立人物名对照表
- 特殊名词首次出现加注
```

### 标点符号规范

```
对话标点：
正确："你说得对。"他点了点头。
正确："你说得对，"他点了点头，"我也这么想。"
错误："你说得对"。他点了点头。

省略号：
正确：……（六个点）
错误：...（三个点）或。。。（句号代替）

破折号：
正确：——（两个汉字长度）
错误：-（一个字符）

引号嵌套：
外双内单："他说：'好的。'"

书名号：
书名/篇名：《书名》
章节名：不加书名号，用粗体或独立一行
```

### 自动校对规则

```
写作时自动检查：

1. 同音字错误
   - 检测上下文语义
   - 标记疑似错误
   - 提供修改建议

2. 形近字错误
   - 常见形近字对照
   - 如：己/已/巳、戊/戌/戍

3. 语法错误
   - 缺主语/谓语
   - 搭配不当
   - 语序混乱

4. 格式错误
   - 引号不匹配
   - 括号不闭合
   - 多余空格/空行

校对输出格式：
原文：[错误句子]
位置：第X章 第Y段
问题：[错误类型]
建议：[修改方案]
```

## 四、内容边界规范

### 内容尺度说明

```
允许范围：
├── 适度情感描写
├── 女性角色的外貌/气质描写
├── 暧昧氛围的营造
├── 情感拉扯和心理博弈
└── 符合剧情需要的亲密场景暗示

描写原则：
├── 以情感为主，感官为辅
├── 留白多于直白
├── 注重氛围和情绪
├── 服务人物塑造和情节推进
└── 避免为描写而描写
```

### 女性角色描写规范

```
外貌描写：
✅ 突出气质特点
✅ 与人物性格呼应
✅ 服务剧情需要
❌ 避免过度聚焦身体部位
❌ 避免物化女性

情感描写：
✅ 注重心理活动和情感变化
✅ 双向奔赴的情感互动
✅ 尊重女性主体性
❌ 避免单向的凝视
❌ 避免强行推进的情感

场景描写：
✅ 注重氛围营造
✅ 适度留白
✅ 情感优先
❌ 避免露骨细节
❌ 避免低俗表达
```

### 禁止内容

```
绝对禁止：
├── 未成年人相关的不当描写
├── 非自愿情节
├── 违法行为的美化
└── 严重违背公序良俗

需要修改：
├── 可能引起争议的表达
├── 过于直白的描写
├── 缺乏铺垫的场景
└── 与人物性格不符的行为
```

## 五、文笔参考标准

### 网文文笔特点

```
核心特征：
1. 简洁有力
   - 一句话说明白一件事
   - 避免冗长修饰
   - 动词优先

2. 画面感强
   - 用具体事物代替抽象
   - 用动作展现状态
   - 用细节营造氛围

3. 节奏明快
   - 短句为主
   - 快速推进
   - 适当停顿

4. 情感直接
   - 不过度委婉
   - 喜怒哀乐明确
   - 读者易代入
```

### 优秀范例

```
场景描写范例：
❌ 她长得很漂亮，身材也很好，很多人都很喜欢她。
✅ 她穿着白色连衣裙从阳光下走来，路过的男生都忍不住回头看。

动作描写范例：
❌ 他很生气，用力拍了一下桌子。
✅ "砰！"他的手掌重重砸在桌上，杯子里的茶溅了一桌。

对话描写范例：
❌ 她说："我不知道该怎么做，很纠结。"
✅ "怎么办……"她咬着嘴唇，"我真的不知道。"

心理描写范例：
❌ 他心里很难过，想起了很多往事。
✅ 熟悉的刺痛感从胸口蔓延。又是这种感觉。
```

### 避免的文笔问题

```
1. 陈词滥调
   - "美若天仙"、"倾国倾城"
   - "心如刀绞"、"痛不欲生"
   → 用具体细节替代

2. 过度修饰
   - 连续使用多个形容词
   - 大量使用"非常"、"极其"
   → 用一个精准的动词/名词

3. 空洞表达
   - "有一种说不出的感觉"
   - "不知为何"
   → 具体描述是什么感觉

4. 信息堆砌
   - 一段话塞入过多信息
   - 无关细节过多
   → 每段一个重点

5. 节奏拖沓
   - 无意义的日常流水账
   - 重复的对话
   → 每场戏都有目的
```

## 六、质量检查清单

### 章节完成检查

```markdown
## 第X章质量检查

### 基础检查
- [ ] 无错别字
- [ ] 标点正确
- [ ] 格式统一
- [ ] 人名一致

### 内容检查
- [ ] 有明确的场景/时间
- [ ] 至少一个冲突点
- [ ] 人物行为符合性格
- [ ] 对话自然不生硬

### 风格检查
- [ ] 句式长短搭配
- [ ] 段落不过长
- [ ] 有画面感
- [ ] 节奏不拖沓

### 改编适配检查
- [ ] 场景可拍摄
- [ ] 对话可朗读
- [ ] 动作可执行
- [ ] 画面可AI生成
```

### 成品检查

```markdown
## 作品完成检查

### 全文一致性
- [ ] 人物前后一致
- [ ] 设定无矛盾
- [ ] 时间线正确
- [ ] 伏笔有呼应

### 整体质量
- [ ] 开篇有吸引力
- [ ] 中段不塌陷
- [ ] 高潮有张力
- [ ] 结局有回味

### 规范符合度
- [ ] 文件存放正确
- [ ] 风格统一
- [ ] 尺度合规
- [ ] 无硬伤
```

## 七、使用说明

### 规范应用时机

```
写作前：
├── 确认文件存放位置
├── 确认写作风格
└── 确认内容边界

写作中：
├── 实时检查错别字
├── 保持风格一致
└── 注意尺度把控

写作后：
├── 完整校对
├── 质量检查
└── 规范对照
```

### 违规处理

```
发现违规内容时：
1. 标记问题位置
2. 说明违规原因
3. 提供修改建议
4. 修改后复查

严重违规：
- 立即停止写作
- 通知作者
- 删除/修改违规内容
- 更新规范提醒
```

## 八、附录

### 常用词汇表

```
描写词汇：
外貌：清秀、妩媚、英气、温婉、冷艳
气质：优雅、飒爽、灵动、沉稳、淡然
声音：清脆、低沉、慵懒、轻柔、沙哑
眼神：清澈、深邃、凌厉、温柔、淡漠

动作词汇：
看：瞥、盯、凝视、打量、端详
走：踱步、穿行、漫步、疾步、踉跄
说：低语、呢喃、嗤笑、冷哼、叹息
```

### 敏感词替代表

```
需替代的表达：
过于直白 → 用暗示/氛围替代
露骨词汇 → 用诗意/含蓄表达
不当描述 → 用情感/心理描写
```
