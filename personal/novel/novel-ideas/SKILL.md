# 小说创意生成器

帮助用户生成结构完整、可执行性强的小说创意方案。

## 工作流程

### 步骤1：理解需求（使用 AskUserQuestion 工具）

如果用户未指定类型，使用 AskUserQuestion 工具询问：

```
问题：请选择创意类型？
选项：
- 玄幻/奇幻：修仙、魔法、异世界等
- 都市：现代都市生活、职场、校园等
- 科幻：未来世界、太空、科技等
- 言情：爱情故事为主
- 悬疑/推理：推理破案、悬疑解谜
- 历史：历史背景故事
- 随机：不限定类型，随机生成
```

### 步骤2：创意发散

生成3-5个不同方向的核心创意

### 步骤3：结构展开

为每个创意补充完整的框架要素

### 步骤4：用户选择（使用 AskUserQuestion 工具）

输出创意后，让用户选择感兴趣的方向：

```
问题：您对哪个创意最感兴趣？
选项：
- 创意一：[标题简述]
- 创意二：[标题简述]
- 创意三：[标题简述]
- 创意四：[标题简述]
- 重新生成：换一批创意
- 混合创意：尝试融合多个创意元素
```

## 输出格式

为每个创意方案输出以下结构：

```
## 创意一：【标题】

**核心梗**：一句话概括最吸引人的卖点

**简介**：100-200字的故事概述，包含开篇悬念和核心矛盾

**主角设定**：
- 姓名：
- 身份/职业：
- 核心特质：性格亮点与致命缺陷
- 成长弧线：人物如何变化

**核心冲突**：
- 外部冲突：主角面对的外在障碍
- 内部冲突：主角内心的挣扎
- 冲突升级点：故事如何层层递进

**结局走向**：
- 情感基调：悲剧/喜剧/开放
- 核心主题：故事探讨的深层命题
- 结局暗示：大致方向（具体结局留白）

**亮点卖点**：这个故事为什么值得一写
```

## 创意生成原则

### 类型适配

根据用户指定类型调整创意风格：
- **玄幻/奇幻**：世界观设定、修炼体系、命运与抗争
- **都市**：现实感、社会议题、情感共鸣
- **科幻**：技术奇点、人性拷问、未来社会
- **言情**：情感拉扯、身份差异、成长救赎
- **悬疑**：信息差、反转设计、真相层层剥开
- **历史**：时代冲突、历史假设、人物命运

### 创意多样性

3-5个创意应覆盖不同方向：
- 题材角度的差异（如：时间穿越 vs 空间异能 vs 身份互换）
- 风格基调的差异（如：热血爽文 vs 虐心催泪 vs 轻松治愈）
- 核心矛盾的差异（如：生存危机 vs 身份认同 vs 情感抉择）

### 创意质量标准

- **新颖性**：避免陈词滥调，有独特的反转或组合
- **可执行性**：有明确的展开方向，能支撑长篇或中篇
- **情感共鸣**：有能打动读者的核心情感点
- **扩展空间**：留有足够的伏笔和支线空间

## 示例

**用户输入**：给我创意，科幻类型，关键词：记忆

**输出**：

## 创意一：【记忆贩卖师】

**核心梗**：一个能提取和贩卖记忆的人，发现自己卖出的记忆正在被篡改。

**简介**：2089年，记忆成为可交易的商品。主角林深是一名持证记忆贩卖师，专门帮人提取痛苦记忆出售。直到某天，一位客户找上门——他买来的记忆与林深卖出的版本完全不同。林深开始追查，发现一个庞大的记忆篡改计划，而他自己卖出的每一段记忆，都可能成为操控他人的武器。

**主角设定**：
- 姓名：林深
- 身份：持证记忆贩卖师，曾是记忆研究所的技术员
- 核心特质：冷静专业、内心孤独（对父亲的记忆有缺失）；缺陷是逃避过去、不敢建立深度关系
- 成长弧线：从逃避到面对，从贩卖记忆到守护记忆的真实性

**核心冲突**：
- 外部冲突：记忆黑市组织想要灭口；客户记忆被篡改引发连锁危机
- 内部冲突：要不要恢复自己被删除的童年记忆？真相可能比遗忘更痛苦
- 冲突升级：发现父亲的失踪与记忆贩卖技术的起源有关

**结局走向**：
- 情感基调：偏悲壮但有希望
- 核心主题：记忆定义了我们是谁；遗忘是一种保护还是一种背叛
- 结局暗示：林深选择公开真相，代价是失去自己的部分记忆

**亮点卖点**：记忆经济的社会想象 + 个人身份认同的哲学命题 + 悬疑追查的紧凑节奏
