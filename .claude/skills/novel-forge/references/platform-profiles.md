# 平台特性档案

## 番茄小说

### 读者画像
- 年龄: 18-35岁为主
- 阅读场景: 通勤、睡前、碎片时间
- 阅读设备: 手机
- 阅读习惯: 快速滑动、3秒判断是否继续、对话驱动型阅读
- 付费意愿: 免费为主（广告模式），追求即时满足感

### 内容规范
- 句均字数: ≤15字（强制）
- 对话占比: ≥60%
- 段落长度: 每段不超过3行（手机屏幕约60字）
- 开篇钩子: 前100字内必须出现（前3行定生死）
- 节奏要求: 每800字至少一个小冲突、信息揭露或反转
- 章节结尾: 必须有悬念（cliffhanger）
- 禁用: 大段心理描写、大段环境描写、超过2句的旁白

### 热门题材
- 婚恋+打脸（离婚后前夫跪求复合、豪门错嫁）
- 重生+复仇（重生后手撕渣男渣女）
- 甜宠+误会（冷面总裁其实暗恋已久）
- 悬疑+反转（身边最亲的人是凶手）

### 标题风格
- 悬念型: "结婚三年，我才发现丈夫的秘密"
- 反转型: "被全家抛弃后，我成了首富"
- 冲突型: "闺蜜抢了我的未婚夫，我抢了她的命"

## 知乎严选

### 读者画像
- 年龄: 22-40岁为主
- 阅读场景: 深度阅读时段（晚间、周末）
- 阅读设备: 手机为主，也有PC端
- 阅读习惯: 愿意接受较慢的开头，但要求逻辑自洽和文笔质量
- 付费意愿: 有付费习惯（盐选会员），对质量有更高期待

### 内容规范
- 句均字数: ≤20字
- 对话占比: ≥40%
- 段落长度: 每段不超过5行
- 开篇钩子: 前200字内必须建立核心冲突或悬念
- 节奏要求: 可以更从容，但每1500字内必须有推进
- 文笔要求: 比番茄更注重语言质量，允许更多文学性表达
- 叙事深度: 可以有更多心理描写和环境描写，但必须服务于情节

### 热门题材
- 职场+权谋（办公室政治、商业博弈）
- 悬疑+社会（反映社会议题的悬疑故事）
- 情感+人性（深度探讨人际关系的复杂性）
- 脑洞+反转（设定新奇、反转出人意料）

### 标题风格
- 叙述型: "我在ICU门口，听到了丈夫和小三的对话"
- 设问型: "为什么那个从不加班的同事，升职最快？"
- 场景型: "婆婆去世那天，我在她枕头下发现了一封信"
